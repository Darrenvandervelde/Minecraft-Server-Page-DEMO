# Minecraft Server Demo

Created with a **live background MP4 video**, featuring a **glass-style container** with the scrollbar hidden for a clean look.

![Minecraft Server Demo](ProjectImg.png)

## About This Project

This project is a **demo landing page for a Minecraft server**, designed to showcase the server in a visually appealing way. It demonstrates:

- **Live background video** for immersive presentation.
- **Glassmorphism design** for content containers.
- **Hidden scrollbar** for a clean user experience.
- **Responsive layout** that adapts to different screen sizes.

The main goal of this project is to provide a **modern, attractive landing page** for Minecraft communities or server listings, serving as a foundation for further development.
